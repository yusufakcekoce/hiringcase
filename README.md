# SEARCH PORTAL

## Run it on your computer

#### Clone the project

```bash
  git clone https://github.com/yusufakcekoce/hiringcase
```

#### Go to the project directory

```bash
  cd src & cd data
```

#### Install required packages

```bash
  npm install
```

#### Run the server

```bash
  json-server --watch mockData.json
```

#### And run the application

```bash
  npm start
```
